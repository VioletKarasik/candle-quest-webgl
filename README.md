# \# Candle Craft Game

# 

# \## Описание проекта

# Это онлайн-игра в жанре квеста-главоломки для бренда свечей \[Witch Craft Shop](https://witch-craft-shop.ru).  

# Игроки проходят серию мини-загадок и интерактивных действий, собирая ингредиенты, взаимодействуя с объектами и создавая свои виртуальные свечи.  

# Игра выполнена в веб-версии и ориентирована на прямой запуск в браузере.  

# 

# \*\*Цель игры:\*\* геймифицировать процесс получения товаров или промокодов бренда, вовлекая аудиторию в интерактивное взаимодействие и повышая вовлечённость.

# 

# \## Идея визуала и стилистика

# Визуальный стиль игры вдохновлён популярной игрой \*Rusty Lake\*: минималистичный, но атмосферный интерфейс с темными и мягкими оттенками, создающими магическую атмосферу.  

# Такой подход позволяет:

# \- Сосредоточить внимание игрока на интерактивных объектах и мини-играх.  

# \- Сделать мир игры лаконичным, без лишних элементов, упрощая навигацию.  

# \- Подчеркнуть тематику свечей и магических ингредиентов через цветовую палитру и анимации объектов.

# 

# \## Геймплейные особенности

# \- Игрок собирает предметы только в правильной последовательности или при выполнении условий (например, книги надо расположить в правильном порядке).  

# \- Мини-игры включают последовательные нажатия, головоломки с предметами, смешивание ингредиентов в котле.  

# \- Собранные предметы добавляются в инвентарь и могут быть использованы для решения новых задач.  

# \- Используется визуальная обратная связь: предметы подсвечиваются, появляются подсказки и предупреждения.  

# \- Фоновая музыка и звуковые эффекты усиливают атмосферу: звук сборки предметов, ошибочных действий и завершения игры.  

# \- После успешного прохождения мини-игр игрок получает бонусы бренда, например, промокод на покупку свечей.

# 

# \## Почему выбрано именно так

# \- \*Rusty Lake\*-подобный визуал создаёт таинственную и магическую атмосферу, не перегружая интерфейс.  

# \- Мини-игры интерактивные, но простые, чтобы любой игрок мог пройти их без затруднений.  

# \- Веб-формат позволяет легко делиться игрой с аудиторией и запускать прямо в браузере.  

# \- Геймификация и интерактивность повышают вовлечённость аудитории, делая процесс знакомства с брендом увлекательным.

# 

# \## Сборка и запуск

# 1\. Игра полностью загружается в браузере без установки.  

# 2\. Рекомендуемое разрешение: 1920x1080, чтобы интерфейс отображался корректно.
